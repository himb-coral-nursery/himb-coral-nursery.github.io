About this dashboard
HIMB Coral Nursery light monitoring

Equipment

  • 2× Odyssey integrating PAR sensors with wipers
  • Sensor names: MARINARA, ALFREDO
  • 1× LI-COR LI-1500 (calibration cross-check, deployed occasionally)

Deployment

  • Depth: 1 m below surface
  • Location: HIMB Coral Nursery, Coconut Island, Kāneʻohe Bay
  • One Odyssey sensor deployed at a time (active sensor shown in header)

Sampling

  • Interval: 5 minutes
  • Wipers: clean optical face every 15 minutes, on the hour
  • Data retrieved: first week of each month

Calibration

  • Calibration coefficients applied in logger software (Odyssey software)
  • LI-COR deployed alongside Odyssey periodically as ground truth
  • Files flagged _CAL in dataset for traceability

Data flow

  • Loggers pulled, CSVs exported
  • Files uploaded via private upload tool (auth required)
  • Data stored in Firebase Realtime Database
  • Dashboard reads data on page load
